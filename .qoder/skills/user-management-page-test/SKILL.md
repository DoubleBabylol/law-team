# 用户管理页面功能测试

## 测试目标

## 测试环境要求

- 前端服务运行在 http://localhost:5173
- 后端服务运行在 http://localhost:8080
- 使用管理员账号登录（如：张三/123456）

## 常见问题

1. **角色标签不显示**：检查后端 `UserService.list()` 是否加载了角色数据
2. **机构列表为空**：检查后端 `/api/branch/list` 接口是否正常
3. **表单验证失败**：检查各字段是否符合校验规则
4. **提交无响应**：检查浏览器控制台网络请求状态
